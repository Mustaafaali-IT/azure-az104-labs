# AZ-104 Azure Administrator Portfolio

## Summary

This repository documents my hands-on learning journey for the Microsoft AZ-104: Azure Administrator certification. It serves as a centralized portfolio of notes, labs, scripts, and real-world implementations as I progress through Microsoft Learn modules and practical exercises.
